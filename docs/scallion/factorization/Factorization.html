  18.  def substitute[A, B](in: (Factorization.this)#Syntax[B], original: (Factorization.this)#Syntax[A], subst: (Factorization.this)#Syntax[A], elim: Boolean = true): (Factorization.this)#Syntax[B]

    Substitutes a Syntax with another one.

    Substitutes a Syntax with another one.

    in

    Syntax in which the substition(s) will happen.

    original

    Syntax which will be removed.

    subst

    Syntax to use as replacement.

    elim

    Indicates whether original must be completely eliminated from returned syntax (see example 2).

    Definition Classes
    Substitution
    Examples: 

    1. lazy val grammar: Syntax[Boolean] = recursive( epsilon(true) | grammar )
lazy val result: Syntax[Boolean] = recursive( epsilon(false) | result )
substitute(grammar, epsilon(true), epsilon(false))  === result
      2. , 

    2. lazy val grammar: Syntax[Boolean] = recursive( (epsilon(true) ~ grammar).map{ case a ~ b => a && b} )
lazy val substitute: Syntax[Boolean] = recursive( (epsilon(false) ~ grammar).map{ case a ~ b => a || b} )
substitute(grammar, grammar, substitute, false) === substitute
substitute(grammar, grammar, substitute, true)
=== recursive( (epsilon(false) ~ substitute).map{ case a ~ b => a || b} )
    Note

    Due to some implementation details, this function might return unexpected results when working with mapped syntaxes:

    val transform = (c: Char) => c.asDigit
val grammar = epsilon('1').map(transform) ~ epsilon(2)
val expected = epsilon(3) ~ epsilon(2)
substitute(grammar, epsilon('1').map(transform), epsilon(3))
=== grammar   // Left unchanged...
=!= expected  // ... which is not what we expected.
